CrewAI João Moura 2024 11, 12, F A multi-agent orchestration framework built on LangChain, allowing AI agents to collaborate through structured roles, tasks, and flows in real-time applications.

Gandalf Dataset Lakera 2023 9 A dataset of adversarial prompts used in a prompt injection game, available on Hugging Face to train or test prompt-aware classifiers.
GitHub Chris Wanstrath, PJ Hyett, Tom Preston-Werner 2008 8, I A platform for hosting and collaborating on open-source code projects, pivotal in the growth of the AI and developer community.
Google Colab Google 2017 1, 2, I A cloud-hosted Jupyter environment with free GPU access, simplifying AI development without requiring local setup.
Gymnasium Farama Foundation 2022 3 A successor to OpenAI Gym, offering reinforcement learning environments like Lunar Lander, used to demonstrate agent-based learning.
HaluBench Patronus AI 2024 8, 9 A benchmark suite for evaluating hallucination performance in LLMs, forming the basis of the LYNX model.
HiFi-GAN Jihan Kim et al. 2020 5 A high-fidelity generative adversarial network vocoder used to convert spectrograms into natural-sounding speech.
Hugging Face Clément Delangue, Julien Chaumond, Thomas Wolf 2016 1, 2, 8, F, I A collaborative platform and open-source leader providing transformer models, datasets, and tools for building modern AI applications.
Hugging Face Spaces Hugging Face 2021 8 A hosted service to deploy ML models with interactive web UIs using Gradio or Streamlit, used to publish the T5 summarizer.
HumanLayer HumanLayer Project 2024 9 A lightweight SDK that wraps AI workflows with human approval checkpoints, ensuring critical actions are gated by review.
Jupyter Notebooks Jupyter.org Community 2014 1 An interactive coding environment that blends code, documentation, and visualizations, ideal for rapid prototyping in AI.
K-Means Clustering MacQueen (1967) / Scikit-learn 1967 / 2010 3 A clustering method applied to superhero features like alignment and PCA Power Score to uncover natural groupings in unlabeled data.
Keras François Chollet 2015 4, 6 A user-friendly neural network API built for rapid experimentation and now the high-level interface of TensorFlow.
LIAR Dataset William Yang Wang 2017 8 A dataset of political statements with labeled truth ratings, used to train and benchmark fact-checking models.
LLaMA 3 Meta AI 2024 1 A state-of-the-art open-source language model designed for reasoning, coding, and content generation, accessible via Hugging Face.
LYNX Patronus AI 2024 9 A hallucination evaluation model trained on real-world benchmarks like HaluBench, designed to detect factual drift and ungrounded model output.
LangChain Harrison Chase 2022 1, 11, 12, 2 An open-source framework for building LLM-based applications with support for chaining models, prompts, and tools.
Librosa Brian McFee 2015 5, 6 A Python library for analyzing and extracting features from audio signals, widely used in music information retrieval and voice analysis workflows.
MIT License Massachusetts Institute of Technology 1988 12 A permissive open-source license that allows reuse, distribution, and modification with minimal restrictions‚Äîwidely used in AI tooling.
MLflow Databricks 2018 10 An open-source platform for managing the ML lifecycle, including experimentation, reproducibility, and deployment across governance workflows.
MNIST Yann LeCun et al. 1998 4 A benchmark dataset of 70,000 handwritten digits widely used to evaluate image classification and pattern recognition models.
MT-Bench LMSYS, UC Berkeley & Carnegie Mellon 2023 9 A multi-turn benchmark for evaluating LLMs in chatbot settings, comparing contextual coherence, instruction following, and conversational quality across tasks.
Matplotlib John Hunter 2003 1, 3 A powerful data visualization library used extensively for plotting and exploring AI datasets.
Milvus Zilliz 2019 2 A scalable, cloud-native vector database supporting distributed retrieval, positioned as an integration-ready option for hybrid data platforms.
Mistral Mistral AI 2024 11, 2, 9 An open-source LLM hosted on Hugging Face, used via LangChain to generate synthetic plot elements and infer missing data.
Model Cards Hugging Face (inspired by Google AI) 2019 10, 9, F Documentation standards that promote transparency by summarizing the intended use, evaluation, and limitations of AI models.
Multinomial Naive Bayes (MNB) Scikit-learn 2010 3 A probabilistic classifier especially effective on TF-IDF-transformed data, used here to predict gender from structured text.
NumPy Travis Oliphant 2005 1 A foundational library for numerical computing in Python, enabling fast array operations essential for machine learning.
ONNX Microsoft & Facebook 2017 4 An open standard for model exchange, enabling interoperability between deep learning frameworks like PyTorch and TensorFlow.

OpenCRE OpenCRE Contributors 2023 9 A toolset for managing and inspecting AI supply chain metadata, promoting transparency via AI Bills of Materials (BOMs).
OpenCV Intel Research 1999 5, 6 An open-source library for real-time computer vision and video analysis, used here for scene detection and frame-by-frame processing.
Pandas Wes McKinney 2008 1, 11, 2, 3 A library for data manipulation using DataFrames, widely adopted for cleaning and analyzing structured data.
Principal Component Analysis (PCA) Karl Pearson (1901) / Scikit-learn 1901 / 2010 3 A technique for dimensionality reduction, used to condense 160+ power attributes into a single PCA Power Score per superhero.
PyTorch Facebook AI Research 2016 4, 6 A widely adopted deep learning library with dynamic computation graphs, ideal for research, prototyping, and scalable deployment across applications.
Python Guido van Rossum 1991 1 A versatile open-source programming language that has become the de facto standard for AI development due to its simplicity and vast ecosystem.
Random Forest Tin Kam Ho (origin) / Scikit-learn 1995 / 2010 3 An ensemble learning method included in a SHAP-based model explanation section, providing robust predictions and explainability.
Relevance AI Relevance AI Team (Sydney, Australia) 2020 11 An open-source platform for vector search and AI workflows, offering tools for embedding, clustering, and deploying AI-powered decision systems.
Responsible AI Microsoft Research 2022 10 A suite of tools for understanding, diagnosing, and mitigating fairness and interpretability issues in machine learning models.
SHAP Scott Lundberg (Microsoft) 2017 3 An open-source tool for explaining individual model predictions, used to visualize feature importance in random forest classifiers.
SceneDetect Breakthrough AI 2018 5 An open-source tool for automatic scene boundary detection in video, used in conjunction with OpenCV for video segmentation.
Scikit-learn David Cournapeau 2007 1, 3 A machine learning library that provides accessible tools for classification, regression, clustering, and more.
SpeechBrain SpeechBrain Team 2021 5 A PyTorch-based toolkit for speech processing, used to extract speaker embeddings in the voice cloning workflow.
SpeechT5 Microsoft Research Asia 2022 5, 6 A transformer-based encoder-decoder model for spoken language tasks including TTS, ASR, and voice conversion.
Stable Diffusion Stability AI 2022 1 A text-to-image generation model that produces visual content from prompts using transformer and diffusion techniques.
Steam Games Dataset Fronkon Games 2023 11 A dataset of over 85,000 titles from Steam, used to inform AI-driven game design through structured prompt-based reasoning.
Superheroes Dataset Claudio Davi 2016 1 A dataset scraped from SuperHeroDB containing character traits and powers, forming the backbone of the data prep examples in this chapter.
T5 Google Research 2020 8 A versatile transformer model that reframes all NLP tasks as text generation, used for summarization in this chapter.
TF-IDF (Scikit-learn) Scikit-learn team 2010 3 A text vectorization technique for transforming superhero descriptions into features used in Naive Bayes models for gender prediction.
TensorFlow Google Brain (Jeff Dean et al.) 2015 4, 6 An open-source framework for building and deploying large-scale machine learning models across servers, mobile devices, and edge environments.
TensorFlow Lite TensorFlow Team 2017 4 A lightweight framework for running TensorFlow models on mobile, embedded, and IoT devices with optimized performance.
TensorFlow.js TensorFlow Team 2018 4 A JavaScript library that runs machine learning models directly in the browser, enabling in-browser inference and training.
TorchServe PyTorch Team 2020 4 A model serving tool for PyTorch, supporting REST APIs, metrics, and multi-model management for production deployment.
Transformers Library Hugging Face 2019 1, 6, 9, F An open-source Python library that simplifies access to pre-trained transformer models for text, vision, and speech applications.
Weaviate SeMI Technologies 2019 2 A graph-aware vector search engine with native support for semantic search, suitable for both text and image embeddings.
What-If Tool Google PAIR 2018 10 An interactive visual interface for exploring counterfactuals and fairness in ML models without writing code.
Whisper OpenAI 2022 5 An open-source speech-to-text model capable of multilingual transcription and translation, used here to transcribe podcast audio.
YOLOv5 Ultralytics 2020 5, 6 A real-time object detection model known for its speed and accuracy, applied here for video annotation of people, animals, and objects.
spaCy Explosion AI 2015 2 An NLP library for tokenization and entity recognition, used here to pseudonymize personal data in AI-generated superhero plots.
